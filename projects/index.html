National Bookshop

  • Tech Stack: MongoDB, Express.js, React.js, Node.js
  • Website: Project Link

The e-commerce web application is designed to provide a seamless online shopping experience for users. The website allows users to register and create their accounts, where they can log in and start browsing the available products. The website also has an admin panel that allows the admin to add new products and manage the inventory.

When users find products they like, they can add them to their cart and proceed to checkout. During the checkout process, the website calculates the shipping fee based on the user's location. If the user has a voucher code, they can enter it to get a discount on their purchase.

After the user confirms the order, the website updates the sold and stock quantities of the products in real-time. The user can also view their transaction history and cancel an order if necessary. They can also change their password or update their account information.

Additionally, the website has a wishlist feature that allows users to save products they are interested in for future purchase. Users can also leave reviews for products they have purchased, which are displayed on the product page.

Overall, the e-commerce web application provides a user-friendly interface and efficient order processing system, ensuring a positive shopping experience for customers.
